Teaching

Since 2016, Prem and I have taught a course in which we rapidly introduce an array of software engineering tools, ranging from static analysis, to version control and continuous integration, to Docker and cloud deployment. They are all taught in combination with their theoretical foundations and histories (Prem's focus), and introduced and explained in weekly discussion sections with corresponding assignments (my focus). This course has been highly popular among CS students, for one because it provides a window into the many tools and aspects of real-world software engineering, but even more so (I hope) because it teaches the ability to quickly adopt, contextualize and understand new tools. I think that ability is pivotal to good software engineering, much more so than knowing any one tool by heart.
